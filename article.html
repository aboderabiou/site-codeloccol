C'EST QUOI HTML

HTML est un langage informatique utilisé pour création des pages web ou des sites internets. le sigle HTML signifie HyperText Markup Language, ce qui signifie en français "langage de balisage d'hypertexte". Cette signification HTML permet de réaliser de l'hypertexte à base d'une structure de balisage.

COMMENT UTILISER LE HTML

  1. Télécharger et installer un éditeur de code dans votre machine, par exemple le sublime text,visuel studio code.
  2. Créer un fichier HTML en mettant de point par exemple index.html sur votre éditeur de code, directement le codes s'affichera.
  3. Remplissez la partie et la partie du document.
  4. Comprenez comment fonctionne la balise.
  5. Écrivez votre première balise .
  6. Ajoutez le titre de votre choix.
  7. Divisez le texte en paragraphes.

C'EST QUOI CSS

Le CSS ou feuilles de style,est un langage informatique qui comprend du code qui permettant de gérer le design d'une page HTML. autrement dit il est utilisé pour maquiller les fichiers HTML

L'avantage de l'utilisation d'un fichier CSS pour la mise en forme d'un site de modifier tous les titres du site en une seule fois en modifiants une seule partie du fichier CSS. Sans ce fichier CSS, il serait nécessaire de modifier chaque titre de chaque page du site.

A QUOI SERT CSS

  1. Mettre la couleur au texte en utilisant par exemple "color: blue;ou bien "background-color: red;".
  2. Modifier la police du texte en utilisant "font-family: italic".
  3. Définir la taille du texte en utilisant "font-size: 20px".
  4. Definir les bordures.
  5. Reduire les images.
  6. alligner les textes à gauche ou à droite de votr choix.
